1. Executive Summary

Taltas is an AI-powered recruitment intelligence platform. The Recruiter Portal is the first of three planned frontend platforms (Recruiter → Candidate → Admin), all sharing the same backend API at https://api.taltas.ai/api/v1.

The portal is a Next.js 14 application with 50+ source files, ~7,500 lines of code, featuring a complete dashboard UI with mock data, full API integration layer, authentication with JWT refresh flow, and 25+ public pages matching the marketing HTML mockup.

5. Authentication

Login → POST /auth/login → { accessToken, refreshToken }
→ Store in Zustand auth-store
→ Axios interceptor attaches Bearer token
→ On 401: POST /auth/refresh → retry failed requests
→ On refresh fail: logout → /login

8. Key Features by Page

Reports (/reports)

  • Radar chart: 300×290 SVG, 6-axis, Explorer (blue bold) vs ATS (gray), values at each apex
  • Pareto chart: 440px wide, 14px bar padding, -40° rotated labels, cumulative line + 80% threshold
  • Funnel: SVG conversion visualization
  • Report export: Tiles with generate + schedule workflow

Modals

  • CandidateModal (362 lines): Overview, Conversation, Fit Analysis, Timeline tabs
  • NewExplorerModal (179 lines): Multi-step wizard, also serves as quick edit
  • ExplorerDetailModal: Stats, activity log, pause/resume
